Features

Got questions about Emperor's capabilities? You've come to right place!

General Features

Some of Emperor's features are broad. They are listed here.

Multi-Everything

Multi-project, multi-user and multi-purpose.

Fully Internationalizable

Ready for porting into the language of your choice via a simple text file.

Search It All

Search is major feature. Tickets, comments, history and timelines are all searchable, facetable and filterable.

Common Knowledge

Use Markdown everywhere. No need to learn proprietary formatting.

Technology

Curious about what Emperor is made of?

Easy Setup

Only JVM and MySQL required for installation, all dependencies included.

Best of Breed Core

Written in Scala using the Play Framework.

Well-Tooled Tech

Uses MySQL and ElasticSearch for storage. Use existing tools for backups. Search indexes can be recreated from MySQL data at any time.

Security

Safe, Secure and Isolated

Secure Passwords

Passwords are stored using bcrypt so you don't have to worry about being in the next list of released passwords.

Permission Schemes

Create permissions schemes and apply them to any or all of your projects. Use them to hide projects from specific users or groups. Don't want a client to see other projects? No problem!

Revokable User Tokens

API, tool and third-party access can be controlled by user-generated tokens that are easily revokable at any time.

Communication

Emperor helps keep your teams connected and up-to-date.

Timelines

When users do work in Emperor it's added to a global timeline that are visible to other team members.

Email Notifications In Progress

Important changes can be configured to generate notification emails, keeping offline team members in the loop.

 

Ticketing

Tickets keep up with the work that needs doing.

Tickets

Create tickets to track bugs, improvements or milestones. You can even create your own ticket types.

History

All changes to tickets are tracked, along with who performed them.

Linking

Keep up with relationships between tickets by linking. Linking can be used to group tickets into a milestone, to show that one ticket is blocked by another or just to relate two tickets because they similar.

Project Management

Emperor is more than ticketing. It's also a great tool for project management.

Customizable Workflow In Progress

Each project can have a custom workflow that is designed just for it's needs.
