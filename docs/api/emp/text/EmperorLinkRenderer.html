EmperorLinkRenderer

class EmperorLinkRenderer extends LinkRenderer

Implements a customer LinkRenderer that interprets wiki-style links — those surrounded with double brackets — and attempts to turn them into something useful. Currently groks @ usernames and things that look like tickets. Ticket or users that do not actually exist will show up with the text-error class.
